# 网盘链接提取器

一个纯前端的百度网盘、夸克网盘分享文本清洗工具。把网盘分享出来的整段文本粘贴进去，会自动提取为：

```text
文件名
链接：https://...
提取码：xxxx
```

## 本地使用

直接用浏览器打开 `index.html`。

## 发布到 GitHub Pages

1. 新建 GitHub 仓库，把本目录文件上传到仓库根目录。
2. 进入仓库 `Settings` -> `Pages`。
3. `Build and deployment` 选择 `Deploy from a branch`。
4. Branch 选择 `main` 和 `/root`，保存。

几分钟后 GitHub 会生成一个可访问的网址。
